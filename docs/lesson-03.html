Lesson 3: Files & Folders

Create, copy, move, and delete files and folders from the command line.

Creating Folders

mkdir FolderName — creates a new folder (same on all platforms).

Create multiple at once: mkdir breakfast lunch dinner

Creating Files

type nul > filename.txt — creates an empty file in PowerShell.

touch filename.txt — creates an empty file.

Writing & Reading Files

echo "text" > filename.txt — writes text to a file (overwrites). Same on all platforms.

cat filename.txt — reads and prints a file's contents. Same on all platforms.

⚠️ The > operator overwrites the file. Use >> to append instead.

Copying, Moving, Deleting

cp source.txt dest.txt — copies a file. Same on all platforms.

mv old.txt new.txt — renames or moves a file. Same on all platforms.

rm filename.txt — deletes a file. Same on all platforms.

rmdir FolderName — deletes an empty folder. Same on all platforms.

⚠️ There is no recycle bin in the terminal. Deleted files are gone permanently.
