# Lesson 2 Hints

Stuck? These hints will guide you without giving away the answers.

---

## Exercise 1: Find Your Location

<details>
<summary>Hint 1</summary>

You need a command that tells you where you are. Think about what the lesson showed you — it's a short command that prints your current directory.

On Windows, try `cd` with no arguments. On macOS and Ubuntu, the command is `pwd`.

</details>

<details>
<summary>Hint 2</summary>

Look for the output that shows your home folder path. On Windows, this might look like `C:\Users\YourName`. On macOS or Ubuntu, it might look like `/Users/YourName` or `/home/YourName`.

</details>

<details>
<summary>Hint 3</summary>

Remember the special symbol for home folder is `~`. Your home folder is where your user files are stored on the computer.

</details>

---

## Exercise 2: Navigate to Desktop

<details>
<summary>Hint 1</summary>

From your home folder, you need to go into a folder called Desktop. Use the change directory command followed by the folder name.

</details>

<details>
<summary>Hint 2</summary>

On all platforms, the command looks the same: `cd Desktop`. But make sure you're starting from your home folder first.

</details>

<details>
<summary>Hint 3</summary>

After navigating, use the list command to see what's on your Desktop. On Windows, this is `dir`. On macOS and Ubuntu, this is `ls`.

</details>

---

## Exercise 3: Navigate Back and Forth

<details>
<summary>Hint 1</summary>

Start by going to your Desktop using `cd Desktop`. Then practice going back to your home folder using `cd ~` (or `cd` with no arguments on Windows).

</details>

<details>
<summary>Hint 2</summary>

From home, try going into Documents: `cd Documents`. To go back two levels, you would use `cd ../..`. The `..` means "parent folder" — so going up two levels takes you to your user's root folder.

</details>

<details>
<summary>Hint 3</summary>

After using `cd ..` or `cd ../..`, use the "where am I" command to confirm your location. This helps verify you ended up where you expected.

</details>
