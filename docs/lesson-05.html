Lesson 5: Challenge — Build a Recipe Collection

Put everything together. Build a folder structure using only the terminal. The checklist on the right tracks your progress automatically.

The Goal

Create a my-recipes folder with three category folders, each containing a recipe file.

my-recipes/
├── breakfast/
│   └── favorite.txt
├── lunch/
│   └── favorite.txt
└── dinner/
    └── favorite.txt

Hint: Step by Step

  1. Create the my-recipes folder: mkdir my-recipes
  2. Create the subfolders: mkdir my-recipes/breakfast my-recipes/lunch my-recipes/dinner
  3. Create the subfolders: mkdir my-recipes/breakfast my-recipes/lunch my-recipes/dinner
  4. Create files: type nul > my-recipes/breakfast/favorite.txt (repeat for lunch, dinner)
  5. Create files: touch my-recipes/breakfast/favorite.txt my-recipes/lunch/favorite.txt my-recipes/dinner/favorite.txt
  6. Add content: echo "Pancakes!" > my-recipes/breakfast/favorite.txt
  7. Verify: ls -R my-recipes
