Lesson 2: Navigating Folders

Learn to move around your computer using the terminal. You'll use these commands every time you open a terminal.

Where Am I?

When you open a terminal, you start in a folder. To see which folder you're in:

cd — with no arguments, prints your current location.

pwd — short for "print working directory."

The location shown is called your current working directory.

Listing Files

dir — lists all files and folders in your current location.

ls — lists all files and folders in your current location.

Useful flags:

dir /a — show hidden files too (files starting with a dot).

ls -a — show hidden files too. ls -la adds size and date details.

Moving Between Folders

cd FolderName — enter a folder (same on all platforms).

cd .. — go up one level to the parent folder (same on all platforms).

cd ~ — go to your home folder. The ~ symbol always means "home."

Windows alternative: cd $env:USERPROFILE

Tip: Press Tab to auto-complete folder names as you type.

Relative vs Absolute Paths

A relative path starts from where you are now: cd Documents

An absolute path starts from the root: cd C:\Users\YourName\Documents

An absolute path starts from the root: cd /Users/YourName/Documents
